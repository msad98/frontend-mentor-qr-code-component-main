# QR Code Component

A simple QR code component built with **HTML and CSS** as a practice project.

This project is based on the **Frontend Mentor – QR Code Component** challenge and focuses on layout, spacing, and basic styling.

## Preview

![QR Code Component Preview](./preview.jpg)

## What I built

- A centered QR code card
- Clean, minimal layout
- Responsive-friendly structure
- Styled using plain CSS (no frameworks)

## Built with

- HTML5
- CSS3

## Purpose

This project was created as a **simple example** to practice:
- Basic HTML structure
- CSS layout and spacing
- Working from a design reference
